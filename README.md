# HomeNetflix
## Requirments
Current system supported :
* Ubuntu
* Debian
## Install
First you have to use sudo
```bash
sudo su
```
Then, clone the repository :
```
git clone https://github.com/MATMAF/HomeNetflix.git
cd HomeNetflix
```
Now execute the script
```
chmod +x ./install.sh
./install.sh
```
